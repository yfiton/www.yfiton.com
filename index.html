Simple one-liner for notifications

$> yfiton --notify-with pushbullet -Pbody="Hello!"

[INFO ] Notification sent in 951 ms by using pushbullet

$> yfiton --notify-with twitter -Pstatus="So simple!"

[INFO ] Notification sent in 668 ms by using twitter

Cross-platform

Yfiton is a true multi-platform solution, supporting Windows, Mac OSX, Linux and Unix. No complex configuration neither system specific packages are required.

Yfiton is open source.

It's hosted, developed, and maintained on GitHub.
